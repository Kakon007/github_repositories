## Features
- Get data from API
- Shimmer Effect
- Pagination
- Filter data
- Can refresh data only once in every 30 mins
- Launch url
- Use Hive as local DB
- Responsive UI
- App's flavor
- Central API call error handling
- GetX
- Network Service
- Clean codes
